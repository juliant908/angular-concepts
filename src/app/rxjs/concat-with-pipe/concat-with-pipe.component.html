RXJS Operators

Operators are RXJS tools that can transform the data being passed through the data stream

Operator concat example

const observable1 = of(1,2,3);
const observable2 = of(4,5,6);
observable1.pipe(concatWith(observable2).subscribe((value) => console.log(value));

concatWith is actually not a completely new operator. It's only meant to replace the concat operator which is currently marked as deprecated and will be removed in v8. There's one subtle difference between the two, though. concatWith only accepts inputs of type ObservableInput, whereas concat can also take a scheduler.
