RXJS Operators

Operators are RXJS tools that can transform the data being passed through the data stream

Operator single example

The only value that is divisible by 2 is:

{{ onlyValue }}

The single operator takes the values from the observable and returns only if ONE value matches the condition passed.

It can return the following responses

MonoTypeOperatorFunction: A function that returns an Observable that emits the single item emitted by the source Observable that matches the predicate.

NotFoundError Delivers a NotFoundError to the Observer's error callback if the Observable completes before any next notification was sent.

SequenceError Delivers a SequenceError if more than one value is emitted that matches the provided predicate. If no predicate is provided, it will deliver a SequenceError if more than one value comes from the source.

EmptyError Delivers an EmptyError if no values were nexted prior to completion.
