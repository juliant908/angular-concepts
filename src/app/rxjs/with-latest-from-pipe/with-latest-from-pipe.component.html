RXJS Operators

Operators are RXJS tools that can transform the data being passed through the data stream

Operator withLatestFrom example

The withLatestFrom uses a main observable that takes values from secondary observables and generates a single result

@if(result$ | async; as result){

In this case the observable emits the value when the second input has any changes (which is the main observable) and takes the LATEST value FROM the secondary observable (first input).

Is evident that you can change the value of the first input and the value emitted doesn't change because the principal observable doesn't emit a new value.
