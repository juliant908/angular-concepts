RXJS Operators

Operators are RXJS tools that can transform the data being passed through the data stream

Operator concat example

const observable1 = of(1,2,3);
const observable2 = of(3,4,5,6);
concat(observable1, observable2).subscribe((value) => console.log(value));

The concat operator merges two (or more) observables into one observable, unlike merge, which fires emissions as they come, concat waits for the previous Observable to complete before it starts the next one.
