RXJS Operators

Operators are RXJS tools that can transform the data being passed through the data stream

Operator merge example

const observable1 = of(1,2,3);
const observable2 = of(3,4,5,6);
merge(observable1, observable2).subscribe((value) => console.log(value));

The merge operator merges two (or more) observables into one observable that emits the values from both observables in the order they are emitted.
